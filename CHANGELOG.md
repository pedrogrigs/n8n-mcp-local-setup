# Changelog

## 0.1.0

Initial public release.

- Cross-platform local installer for `n8n-mcp`
- HTTP mode on `127.0.0.1`
- Local bearer token generation
- Windows Scheduled Task autostart
- macOS LaunchAgent autostart
- Linux systemd user service with XDG fallback
- Codex MCP configuration
- Claude Code MCP configuration
- Hosted `api.n8n-mcp.com` migration
- `/health` and `tools/list` validation
